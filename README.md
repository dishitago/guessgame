# 🎮 Number Guessing Game

A simple Java console-based game where the player tries to guess a randomly generated number between 1 and 50.

## 📌 Features

- Random number generation using Java's `Random` class
- User input using `Scanner`
- Hints provided after each guess:
  - Too High
  - Too Low
- Tracks total number of attempts
- Winning message when the correct number is guessed

## 🛠️ Technologies Used

- Java
- Scanner Class
- Random Class
